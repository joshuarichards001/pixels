# Ten Thousand Pixels

A real-time collaborative pixel art canvas where users can draw pixels and see live updates (similar to Reddit's r/place)

- Frontend
  - HTML
  - CSS
  - JavaScript
  - HTML5 Canvas
  - Hosted using Vercel
- Backend
  - Go
  - Websocket's
  - Hosted using fly.io
- Data storage
  - Redis
  - Hosted using Redis
